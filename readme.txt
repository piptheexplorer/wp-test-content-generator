Generate fake test posts for posts, pages, and custom post types, with featured images and taxonomy terms.

== Description ==

WP Test Content Generator adds a page under Tools > Test Content Generator.

Features:

* Select any supported post type.
* Choose how many items to create.
* Create published, draft, or pending content.
* Generate varied titles, excerpts, and formatted body content.
* Generate a placeholder featured image for each post.
* Use one selected Media Library image for every post.
* Randomly assign existing Media Library images.
* Select existing categories or custom taxonomy terms.
* Create new terms during generation.
* Assign all selected terms or randomise them per post.
* Mark generated posts and placeholder images for safe cleanup.

== Installation ==

1. Upload the plugin ZIP through Plugins > Add New > Upload Plugin.
2. Activate WP Test Content Generator.
3. Open Tools > Test Content Generator.
4. Choose a post type, quantity, featured-image source, and taxonomy settings.
5. Click Generate Test Posts.
